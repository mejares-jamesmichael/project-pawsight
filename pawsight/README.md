# PawSight

**PawSight** is a mobile application designed to help pet owners (starting with cats) interpret their pet's body language.

## Features
*   **Offline Library:** A fast, searchable field guide of cat behaviors (SQLite).
*   **AI Assistant:** Chat with an AI agent to analyze behavior (via n8n).
*   **Vet Hotline:** Quick access to emergency resources.

## Getting Started
This project is a starting point for a Flutter application.
